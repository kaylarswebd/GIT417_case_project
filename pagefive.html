These are some properties that are called Navigator Objects that can obtain information using JavaScript.

These Navigator Objects are properties that JavaScript can use that can tell what someone user properties are. These simple commands can tell a lot about the user’s location, computer specs, and about the browser. And that information in the wrong hands is no good.

First there is the appName property, if you look in the console log after in the inspecting mode, I have put this code there and it will tell the name of the web browser that is displaying the page. This is dangerous because it can tell people what browser you are using and that information you do not want to be in the wrong hands.

Next is the appVersion, this property will tell the user what version of the browser you are using. So not only can someone see what browser you are using, but they can also tell what version you are using.

These next two properties to me are a little bit scarier than the first two. the first one that I am going to talk about is the geoLocation. This property is very dangers because people can now see where you are! This obviously does not need explanation as to why it is a bad thing if someone you would not wanting to know this information did.

Another property that I am going to talk about is the onLine property. This is the console log will display with a true or false weather or not the person looking at the page has an internet connect or not.

The next one that I am going to talk about is the platform property. This property if logged via console.log will show the persons operating system that is being used on the client’s computer.

And finally, is the userAgent property. Not only can this property give information about your browser, but it can also get information about your platform name and compatibility as well.

These are all properties that contain very sensitive information about the clients computer and other information that in the wrong hands can really be dangerous if the information is available. If someone would want to get into your computer the would have a really good start if they had all of this information.
